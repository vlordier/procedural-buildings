# Orthodox Church Grammar

## Goal
Create `grammars/orthodox_church` — a Ukrainian/European church with a golden dome,
requiring a new `dome.obj` primitive.

## New Primitive: `dome.obj`
A hemisphere with a lantern (small cupola on top), procedurally generated:

- 16 segments around, 8 rings up
- Radius ~0.5, height ~0.65
- Lantern: small cylinder + smaller hemisphere at the apex
- Base at z=0, centered at (0.5, 0.5) in xy
- Bounding box: (0,0,0)-(1,1,0.75)

Generated by a one-shot script `scripts/gen_dome.py`.

## Church Grammar
```
whiteR = 0.95
whiteG = 0.93
whiteB = 0.88
goldR = 0.85
goldG = 0.75
goldB = 0.20
roofR = 0.25
roofG = 0.40
roofB = 0.35
plot --> S(~1, ~1, 12){church}
church --> split(z){~8 : churchWalls | ~2 : churchRoof | ~2 : domeTower}
churchWalls --> colour(whiteR, whiteG, whiteB){comp(f){front : colour(whiteR, whiteG, whiteB){churchFacade} | left : colour(whiteR, whiteG, whiteB){I(rect)} | right : colour(whiteR, whiteG, whiteB){I(rect)} | back : colour(whiteR, whiteG, whiteB){I(rect)}}}
churchFacade --> split(x){~1 : I(rect) | ~2 : opening(0.2){colour(0.3,0.3,0.4){I(rect)} | nil} | ~1 : I(rect) | ~2 : opening(0.2){colour(0.3,0.3,0.4){I(rect)} | nil} | ~1 : I(rect)}
churchRoof --> colour(roofR, roofG, roofB){S(width * 1.1, depth * 1.1, 2){I(triangle)}}
domeTower --> S(~0.4, ~0.4, ~1){colour(goldR, goldG, goldB){I(dome)}}
```

The `domeTower` sits on top of the roof: a tall narrow cylinder scope
(width 0.4 of the building) with the dome primitive on top.
